## 2. خطوات التشغيل من الصفر

### الخطوة 1: تحميل المشروع

```bash
# إذا كان لديك الكود بالفعل، انتقل لمجلده مباشرة
cd "Desktop/مؤشر التداول /market-lion"

# أو استنسخه من المستودع
git clone https://github.com/razan/market-lion.git
cd market-lion
```

### الخطوة 2: إعداد ملف البيئة

```bash
# نسخ القالب
cp .env.example .env

# فتح الملف للتعديل
nano .env   # أو استخدم أي محرر نصوص
```

**أهم الحقول الإلزامية:**

```env
JWT_SECRET=ضع_هنا_سلسلة_عشوائية_طويلة_64_حرف_على_الأقل
ENCRYPTION_KEY=ضع_هنا_مفتاح_Fernet
```

لتوليد `JWT_SECRET` قوي:
```bash
python3 -c "import secrets; print(secrets.token_hex(32))"
```

لتوليد `ENCRYPTION_KEY`:
```bash
python3 -c "from cryptography.fernet import Fernet; print(Fernet.generate_key().decode())"
```

### الخطوة 3: تشغيل المنصة

```bash
# إعطاء صلاحية التنفيذ (مرة واحدة فقط)
chmod +x start.sh

# تشغيل كل شيء تلقائياً
./start.sh docker
```

سيقوم السكريبت بـ:
1. ✅ التحقق من وجود Docker
2. ✅ إنشاء `.env` من القالب إذا لم يكن موجوداً
3. ✅ بناء جميع خدمات Docker
4. ✅ الانتظار حتى تصبح قواعد البيانات جاهزة
5. ✅ الانتظار حتى تصبح الـ Gateway جاهزة
6. ✅ عرض جميع روابط الوصول

### الخطوة 4: الوصول للمنصة

بعد الانتهاء، افتح متصفحك على:

| الخدمة | الرابط |
|--------|--------|
| 🦁 لوحة التحكم الرئيسية | http://localhost:3000 |
| 🔌 API Gateway (للمطورين) | http://localhost:8000 |
| 📚 وثائق API | http://localhost:8000/docs |
| 🔑 خدمة المصادقة | http://localhost:8001 |
| 📊 Grafana (المراقبة) | http://localhost:3001 |

### أوامر التشغيل الأخرى


## 3. إعداد مفاتيح API

### مفاتيح إلزامية (مجانية)

#### أ) Alpha Vantage — بيانات السوق
1. اذهب إلى: https://www.alphavantage.co/support/#api-key
2. سجّل بريدك الإلكتروني واحصل على مفتاح مجاني فوراً
3. المجاني يتيح: 25 طلب/يوم
4. ضعه في `.env`:
   ```env
   ALPHA_VANTAGE_KEY=مفتاحك_هنا
   ```

#### ب) NewsAPI — الأخبار الاقتصادية
1. اذهب إلى: https://newsapi.org/register
2. سجّل حساباً مجانياً (100 طلب/يوم)
3. ضعه في `.env`:
   ```env
   NEWSAPI_KEY=مفتاحك_هنا
   NEWS_API_KEY=مفتاحك_هنا
   ```

#### ج) FRED — البيانات الاقتصادية الأمريكية
1. اذهب إلى: https://fred.stlouisfed.org/docs/api/api_key.html
2. سجّل حساباً مجانياً (مجاني بالكامل)
3. ضعه في `.env`:
   ```env
   FRED_API_KEY=مفتاحك_هنا
   ```

### مفاتيح التداول الآلي (اختياري — للتداول الفعلي)

#### د) MetaApi — MT4/MT5
1. اذهب إلى: https://metaapi.cloud
2. انشئ حساباً وربط حساب MT4/MT5 الخاص بك
3. احصل على API Token من لوحة التحكم
4. ضعه في `.env`:
   ```env
   METAAPI_TOKEN=توكنك_هنا
   ```

#### هـ) Capital.com
1. افتح حساباً في Capital.com
2. اذهب إلى الإعدادات → API
3. أنشئ مفتاح API جديد
4. ضعه في `.env`:
   ```env
   CAPITAL_COM_API_KEY=مفتاحك_هنا
   CAPITAL_COM_PASSWORD=كلمة_المرور
   ```

### مفاتيح الإشعارات (اختياري)

#### و) Telegram Bot
1. افتح تطبيق تيليغرام
2. ابحث عن `@BotFather`
3. أرسل `/newbot` واتبع التعليمات
4. احصل على التوكن (مثال: `123456789:AABBcc...`)
5. ضعه في `.env`:
   ```env
   TELEGRAM_BOT_TOKEN=توكنك_هنا
   ```
6. للحصول على Chat ID: أرسل رسالة للبوت ثم افتح:
   `https://api.telegram.org/botTOKEN/getUpdates`

#### ز) CoinMarketCap (للكريبتو — اختياري)
1. اذهب إلى: https://pro.coinmarketcap.com/signup
2. الباقة المجانية: 10,000 طلب/شهر
3. ضعه في `.env`:
   ```env
   CMC_API_KEY=مفتاحك_هنا
   ```

#### ح) Glassnode (للبيانات الـ On-Chain — اختياري)
1. اذهب إلى: https://glassnode.com
2. الباقة المجانية محدودة — للاستخدام الأولي كافية
3. ضعه في `.env`:
   ```env
   GLASSNODE_API_KEY=مفتاحك_هنا
   ```

### بعد إضافة المفاتيح

```bash
# أعد تشغيل المنصة لتطبيق التغييرات
./start.sh stop
./start.sh docker
```

## 4. شرح لوحة التحكم

### الشاشة الرئيسية

لوحة التحكم مقسمة إلى أقسام رئيسية:

#### 🟡 شريط الأعلى (Top Bar)
- **اختيار الأصل**: XAU/USD (ذهب) | WTI/USD (نفط) | EUR/USD | وغيرها
- **اختيار الإطار الزمني**: M15 | M30 | H1 | H4 | D1
- **وضع التداول**: آلي تلقائي ↔ يدوي مع إشارات
- **مؤشر حالة الاتصال**: 🟢 متصل | 🔴 منقطع
- **درجة التوافق الكلية**: نسبة مئوية من 0-100%

#### 📊 الشارت الرئيسي
يعرض شارت الشموع اليابانية مع الأدوات التالية (قابلة للتشغيل/الإيقاف):
- **الشموع**: خضراء للصاعدة، حمراء للهابطة
- **المتوسطات المتحركة**: EMA 7, EMA 10, EMA 25, SMA 50, SMA 100, SMA 200
- **Order Blocks**: مستطيلات خضراء/حمراء مع الحجم المالي
- **مناطق العرض والطلب**: مناطق مؤسسية للبيع والشراء
- **خطوط الدعم والمقاومة**: خضراء للدعم، حمراء للمقاومة
- **نماذج السعر**: رسم آلي لكل الأنماط المكتشفة مع الاسم
- **أنماط الشموع اليابانية**: اسم النمط مكتوب بجانبه
- **موجات إليوت**: ترقيم الموجات تلقائياً
- **إشارات Buy Razan / Sell Razan**: إشارات مؤكدة مع صانعي السوق

#### 📋 الجداول الأربعة

**جدول رقم 1 — التحليل الأساسي:**
يعرض كل الأخبار الاقتصادية لحظياً مع:
- الخبر ومصدره
- التوقيت (عالمي + محلي)
- الإطار الزمني المستهدف
- السابق / المتوقع / الفعلي
- تأثيره على الأصل (إيجابي/سلبي/محايد)
- توصية البيع أو الشراء

**جدول رقم 2 — التحليل الفني:**
يعرض كل مدرسة تحليلية (65+ مدرسة) مع:
- اسم المدرسة/المؤشر
- حالتها على 5 إطارات زمنية (M15, M30, H1, H4, D1)
- خانات خضراء للشراء وحمراء للبيع
- نسبة الثقة الكلية في الأسفل

**جدول رقم 3 — السيولة وتدفق الأموال (Whale Tracker):**
يرصد كل صفقة فوق $10,000 مع:
- حجم الصفقة بالفئة ($10K+ | $100K+ | $500K+ | $1M+ | $10M+)
- اتجاه الصفقة (شراء/بيع)
- الجهة/المتداول (إن أمكن التعرف عليه)
- تصنيف النشاط (تجميع/تصريف/تحوط)

**جدول رقم 4 — خطة التداول وإدارة المخاطر:**
يحسب تلقائياً بناءً على رأس مالك:
- نوع الصفقة وإطارها الزمني
- نقطة الدخول المثلى
- وقف الخسارة (Smart SL)
- الأهداف الثلاثة (TP1 | TP2 | TP3) بنسبة ريشيو 1:3 كحد أدنى
- حجم اللوت المناسب لرأس المال
- الرافعة المالية الموصى بها
- نسبة المخاطرة (لا تتجاوز 2% للمبتدئين)
- الأرباح والخسائر المتوقعة

## 6. نظام الاشتراكات والبيع التجاري

### الخطط المتاحة

| الخطة | السعر/شهر | المميزات |
|-------|-----------|----------|
| 🆓 مجانية | $0 | تحليل محدود، بيانات متأخرة 15 دقيقة |
| 🥉 مبتدئ (Starter) | $29 | إشارات لحظية، 3 أصول، 1 حساب تداول |
| 🥈 محترف (Pro) | $79 | جميع الأصول، 3 حسابات، تداول آلي |
| 🥇 VIP | $149 | كل المميزات + ضبط الأوزان + الدعم المباشر |
| 🏢 مؤسسي (Enterprise) | تواصل معنا | FIX API + White Label + Prop Firm |

### إعداد نظام الدفع (Stripe)

1. انشئ حساباً في: https://dashboard.stripe.com
2. اذهب إلى **Developers → API keys**
3. انسخ **Secret key**:
   ```env
   STRIPE_SECRET_KEY=sk_live_...
   ```
4. لإعداد Webhook:
   - اذهب إلى **Developers → Webhooks**
   - أضف endpoint: `https://yourdomain.com/auth/stripe/webhook`
   - انسخ **Signing secret**:
   ```env
   STRIPE_WEBHOOK_SECRET=whsec_...
   ```

### إعداد HyperPay (للمنطقة العربية)

1. تواصل مع: https://www.hyperpay.com
2. احصل على Access Token بعد التسجيل
3. ضعه في `.env`:
   ```env
   HYPERPAY_ACCESS_TOKEN=توكنك_هنا
   ```

### نظام الإحالة (Referral)

كل مستخدم يحصل تلقائياً على رمز إحالة فريد.
عند اشتراك صديقه:
- **المحيل**: خصم 20% على الاشتراك التالي
- **المُحال**: خصم 10% على أول اشتراك

### إدارة الاشتراكات (للمدير)


## 7. بدء التداول الحقيقي بأمان

### ⚠️ تحذير مهم قبل البدء

> **لا تبدأ بالتداول الحقيقي قبل إكمال هذه الخطوات كاملاً.**
> التداول ينطوي على مخاطر. نتائج الـ Backtest لا تضمن الأرباح المستقبلية.

### الخطوة 1: اختبار على حساب تجريبي (Demo)

1. افتح حساباً تجريبياً في Capital.com أو Exness
2. أدخل بيانات الحساب في `.env`
3. شغّل التداول الآلي في وضع **Demo** أولاً:
   ```
   الإعدادات → وضع التداول → Demo
   ```
4. راقب الأداء لأسبوعين على الأقل

### الخطوة 2: إعدادات إدارة المخاطر

في لوحة التحكم → إعدادات المخاطر:

```
نسبة المخاطرة لكل صفقة:  1% (للمبتدئين) — 2% (للمتوسطين)
الحد الأقصى:              5% (لا تتجاوزه أبداً)
أقصى عدد صفقات متزامنة:  3
أقصى خسارة يومية:         3%
أقصى تراجع (Drawdown):    10%
```

### الخطوة 3: تفعيل Circuit Breaker

هذا النظام يحميك تلقائياً:

| المستوى | الشرط | الإجراء |
|---------|-------|---------|
| 1 | 3 خسائر متتالية | توقف 2 ساعة + تنبيه |
| 2 | خسارة يومية 3% | إيقاف اليوم كاملاً |
| 3 | تراجع 10% | تقليل الحجم 50% |
| 4 | تراجع 20% | إيقاف كامل + مراجعة |
| 5 | تراجع 30% | إيقاف دائم يدوي |

### الخطوة 4: تفعيل التداول الحقيقي

```
1. الإعدادات → حساب التداول → أدخل بيانات حسابك الحقيقي
2. الإعدادات → وضع التداول → Live
3. الإعدادات → حجم البداية → ابدأ بأصغر حجم ممكن (0.01 لوت)
4. راقب أول 5 صفقات قبل رفع الحجم
```

### الخطوة 5: مراقبة الأداء

```
# لوحة المراقبة في Grafana
http://localhost:3001

مؤشرات يجب مراقبتها:
- Win Rate (نسبة الربح): يجب أن تكون > 65%
- Average R:R (متوسط الريشيو): يجب أن يكون > 1:2
- Max Drawdown: يجب أن يبقى < 10%
- Profit Factor: يجب أن يكون > 1.5
```

## 8. الأسئلة الشائعة وحل المشاكل

### ❓ لا تظهر لوحة التحكم

```bash
# تحقق من حالة الخدمات
./start.sh status

# اقرأ السجلات
./start.sh logs frontend

# الحل الشائع: أعد البناء
./start.sh stop
./start.sh docker
```

### ❓ رسالة "JWT_SECRET غير صالح"

```bash
# تحقق من .env
grep JWT_SECRET .env

# يجب أن يكون طولاً كافياً وغير افتراضي
# احذف القيمة القديمة وضع قيمة جديدة:
python3 -c "import secrets; print(secrets.token_hex(32))"
```

### ❓ لا تظهر الإشارات

أسباب شائعة:
1. **مفتاح Alpha Vantage فارغ** → أضفه في `.env`
2. **درع الأخبار نشط** → انتظر انتهاء الإيقاف المؤقت
3. **درجة التوافق أقل من 75%** → السوق غير مناسب للدخول الآن (هذا صحيح)
4. **Circuit Breaker مفعّل** → ارجع إلى إعدادات المخاطر وأعد تشغيله يدوياً
